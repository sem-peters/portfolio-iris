Beste beoordelende docent,

Deze website is gemaakt door gebruik te maken van Visual Studio Code.
In VSCode is het mogelijk om een HTML bestand te openen met een Live Server.

Als de HTML bestanden in een niet-server omgeving worden weergegeven (door bijvoorbeeld het
index.html bestand gewoon te openen in de browser, ZONDER liveserver), kan het zijn
dat bepaalde onderdelen van de website niet werken.

Voorbeelden zijn: ontbrekende afbeeldingen, ontbrekende CSS, ontbrekende JavaScript.

Ik zou het daarom waarderen als u het portfolio zou beoordelen in de staat waarin
hij echt zou werken, namelijk in een (lokale) live serveromgeving zoals met VSCode.
